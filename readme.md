# Salt & Sovereignty

A static HTML, CSS, and JavaScript website for **Salt & Sovereignty**, Ashley's podcast, author platform, and digital sanctuary for witchcraft, healing, and self-reclamation.

The site includes a blog, podcast hub, book information for **The Lost Goddess Trilogy**, and an interactive digital altar designed to provide an accessible ritual space for practitioners.

## Features

* Responsive witchcore design
* Podcast homepage and embedded player
* Static HTML blog and grimoire entries
* Interactive Digital Altar

  * Draggable altar objects
  * Resize, rotate, flip, duplicate, and layer objects
  * Lock and glow effects
  * Animated candle lighting and extinguishing
  * Herb and oil dressing system with visual overlays
  * Save, load, and clear altar layouts using browser storage
  * Mobile-optimized altar interface with slide-up cabinet
* Threshold (landing) experience
* Responsive navigation and layout
* Accessibility-minded design

## Technologies

* HTML5
* CSS3
* Vanilla JavaScript (no frameworks)
* GitHub Pages for hosting
* Browser Local Storage for saving altar layouts

## Local Development

Clone the repository and open `index.html` in your browser, or serve the project using any static web server.

No build tools, package manager, or framework are required.

## Hosting

The site is hosted using **GitHub Pages** and consists entirely of static files.
