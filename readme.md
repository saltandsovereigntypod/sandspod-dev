# Salt & Sovereignty

A static, single-page personal brand website for Ashley, creator of the Salt & Sovereignty podcast and author of The Lost Goddess Trilogy.

## Technologies

- HTML for semantic page structure
- CSS for the responsive witchcore visual system and animations
- JavaScript for mobile navigation, scroll reveals, and Netlify form submission feedback
- Netlify Forms for contact form handling

## Local Use

Open `index.html` in a browser, or serve the project with any static file server. The site deploys directly to Netlify with `netlify.toml` publishing the repository root.
